# Contributing to VALTER

Thank you for your interest in building the future of Data Operating Systems!

## Getting Started
1. Fork the repo.
2. Run `just dev` to start the Monorepo environment.
3. Make your changes in `core/` (Rust) or `dashboard/` (React).

## Pull Requests
- Please follow the existing code style.
- Ensure `cargo test` passes.
- Update documentation if you change configuration options.
