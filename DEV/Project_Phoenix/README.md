# Project Phoenix

## Overview
Project Phoenix is a hyper-converged database system designed for high-throughput transactional workloads. It utilizes a custom filesystem-based storage engine.

## Technology Stack
- **Language:** Rust (Edition 2021)
- **Database:** SQLite (Embedded), DuckDB (Analytics)
- **API:** GraphQL (Async-GraphQL)

## Deployment
Running on a distributed cluster of edge nodes.
